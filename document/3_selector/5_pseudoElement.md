# 偽元素選擇器

-   ::開頭
-   舊版的 W3C Spec 沒有將 pseudo-class 和 pseudo-element 區分成 : 和 ::
-   用:開頭寫偽類，大部份瀏覽器還是會支援

## Generated Content Pseudo-elements

-   ::before
-   ::after

## Other Pseudo-elements

-   first-letter
-   first-line
-   placeholder
-   selection
