# 如何添加 CSS 來源

外部樣式表 (External CSS)

```html
<head>
    <link rel="stylesheet" href="External.css" />
</head>
```

內部樣式表 (Internal CSS)

```html
<head>
    <style>
        p {
            color: red;
        }
    </style>
</head>
```

行內樣式 (Inline CSS)

```html
<body>
    <p style="color:red;">Apple</p>
</body>
```
